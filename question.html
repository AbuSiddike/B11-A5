Question-3: What is event delegation in the DOM?

Delegation is a technique in JavaScript where a single event listener is added to a parent element instead of multiple event listeners on individual child elements. The event bubbles up from the target element to the parent, allowing efficient event handling.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism in the DOM where an event starts from the target element and propagates (bubbles) up through its ancestors until it reaches the document root.

Question-5: How do you create, add, and remove elements using JavaScript?
